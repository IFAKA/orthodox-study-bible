# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. Instead, report it privately via email or GitHub's private reporting feature (if enabled).

We will respond within 48 hours and work with you to resolve the issue as quickly as possible.
